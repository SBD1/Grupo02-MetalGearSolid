Título: Code Talker
Descrição: Faça contato e extraia o Code Talker, um idoso que sabe como tratar o patógeno que afeta a Base Mãe. A mansão onde ele está localizado é guardada pela força de pacificação e, além disso, observou-se uma força não identificada operando na área.
Terreno: África central.
Objetivos:
	Principais:
		Extraia o Code Talker
	Secundários:
		Obtenha os diamantes brutos.
		Obtenha material processado da mansão.
		Obtenha os containers com materiais brutos.
